» Installation Instructions
  1. Open wp-content/plugins Folder
  2. Put:
    Folder: wp-serverinfo
  3. Activate WP-ServerInfo Plugin
  4. Refer To Usage For Further Instructions
» Upgrade Instructions
From v1.0x To v1.50
  1. Deactivate WP-ServerInfo Plugin
  2. Open wp-content/plugins Folder
  3. Put/Overwrite:
    Folder: wp-serverinfo
  4. Delete this folder if exists:
    Folder: serverinfo
  5. Activate WP-ServerInfo Plugin
  6. Refer To Usage For Further Instructions
» Usage Instructions
General Usage
  1. Go to 'WP-Admin -> Dashboard -> WP-ServerInfo'
